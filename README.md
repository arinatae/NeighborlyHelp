# 🏡 Соседская помощь (Neighborly Help)

> **Тёплая 2D-игра о добрососедстве, созданная на C# (Windows Forms)**

---

## 📖 Описание

**«Соседская помощь»** — это уютная 2D-игра в жанре повседневного квеста. Вы играете за отзывчивого жителя небольшого двора, который помогает соседям с их мелкими, но важными просьбами. Найдите потерянные очки, отнесите посылку бабушке, соберите рассыпавшиеся продукты — каждое доброе дело приближает вас к званию лучшего соседа!

Игра создана в рамках учебного курса по программированию на C#. Основная цель — демонстрация навыков объектно-ориентированного программирования, работы с Windows Forms и проектирования игровой архитектуры.

---

## ✨ Особенности

| Особенность | Описание |

## 🎮 Геймплей

### Основная механика
1. **Перемещение** — подходите к соседям и объектам во дворе
2. **Диалог** — узнаёте о просьбах и проблемах жителей
3. **Выполнение заданий** — ищете предметы, взаимодействуете с окружением
4. **Прогресс** — новые задания открываются после выполнения предыдущих

### Примеры заданий
- 🔑 **Потерянные ключи** — найти у скамейки и вернуть владельцу
- 📦 **Посылка для бабушки** — забрать в одном месте, доставить в другое
- 🛒 **Рассыпавшиеся продукты** — мини-игра по сбору предметов
- 🌸 **Полить цветы** — найти лейку и ухаживать за клумбой

### Длительность прохождения
⏱️ **~5 минут** для демонстрации всех основных механик

### 🎯 Управление

| Клавиша / Действие | Функция |
|--------------------|---------|
| **WASD / Стрелки** | Перемещение героя |
| **ЛКМ (клик)** | Взаимодействие с объектами/NPC |
| **E** | Открыть диалог / Действие |
| **I** | Открыть инвентарь |
| **Esc** | Пауза / Меню |
